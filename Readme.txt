Squawk source package.

To run:

    1, Have a version of at least Java 1.3 installed (recommend build 1.3.1-b24)
    2, cd run
    3, Execute one of the command lines in the file "example.txt"

To build:

    1, Have a version of at least Java 1.3 installed (recommend build 1.3.1-b24)
    2, If using Windows have MKS Toolkit version 6.2a installed
    3, sh bld.sh

To clean:

    1, sh del.sh
